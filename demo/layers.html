Dropdowns

Some facts about layered elements:

  • Layered elements (dropdowns and popups) could be nested
  • Nested layered elements are creating tree
  • Second click on opening layer element closes the layer
  • Any click outside top layer will close it
  • Element that opening layer is caller
  • Caller is any element that have su-target attribute
  • Any layered element should have an attribute anchor, that tells to caller what to display
  • Different callers can be open the same layered elements, but only if they are not nested by loop

Some more about dropdowns:

  • If dropdown callers width is less then 50 pixels, dropdowns arrow is closer to edge.
